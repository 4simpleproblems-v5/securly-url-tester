Securly URL Tester

A lightweight, proxy-compatible tool to check website blocking status.

 | 

JavaScript Integration

The securly_tester.js file exports a single async function. It is designed to work in any ES Module environment.

2. Import and Usage

Import the function and pass the student email and target URL.

import { checkSecurly } from './securly_tester.js';

// Run the check
async function testSite() {
    const status = await checkSecurly('student@school.edu', 'example.com');
    
    if (status === 'BLOCKED') {
        console.log("This site is blocked!");
    } else if (status === 'ALLOWED') {
        console.log("This site is accessible.");
    }
}

testSite();

3. Configuring the Proxy

By default, the script uses corsproxy.io. To use your own proxy, edit the applyProxy function:

// Inside securly_tester.js
const applyProxy = (targetUrl) => {
    return '/service/' + btoa(targetUrl); 
};
